概述

OpenVPN(https://openvpn.net)是一个开放源码的基于SSL的VPN系统，比基于PPTP VPN最大的优势在于仅仅使用UDP/TCP协议，支持从NAT设备后的连接。

优点是可以设置只有通往指定网络的流量才会走vpn隧道

公司采用账号密码+ca方式授权验证登录

 

获取证书及配置文件

从管理员处获取open-vpn 客户端的证书及配置文件.

也可以从本帮助页面自助下载.

## 安装客户端openvpn-install-2.4.8-I601-Win10.exe

​
​
##将获取的配置文件复制到openvpn 的config 目录,
#目录一般为: C:\program files\openvpn\config  具体位置为安装客户端时的目录
配置文件为:
├── ca.crt
├── client.ovpn 
​
###
#启动
双击桌面的OpenVPN GUI图标:在这个openvpn-gui.exe文件上单击鼠标右键,选择"以管理员身份运行"(Run as Administrator)
在任务栏右下角通知栏中找到OpenVPN的图标，右键
点击连接，然后点击连接
##输入账号密码
连接过程会出现很多日志，连接成功后，右下角会提示连接成功
​
##单击openvpn-gui，在右下角openvpn-gui  ,单击"connect"可以连接vpn 屏幕窗口内出现大量信息，最后出现 Initialization Sequence Completed 的信息，右下角弹出窗口，显示获得的IP地址信息
​
#连接成功

​
#启动
sudo  systemctl   start    openvpn@client.service
#开机自启动
sudo  systemctl  enable    openvpn@client.service
